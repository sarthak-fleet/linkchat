## Context & Scope

Karte is an AI-powered link-in-bio that replaces a static list of links with four interactive surfaces: Chat, Encyclopedia, Newspaper, and Roast. The product lives in an emerging category ("AI link-in-bio") that has no established SEO landscape yet — which is an opportunity. This plan defines 20 content cluster pieces (glossary terms + blog articles) that can own definitional and pain-point searches, drive organic traffic to `karte.cc`, and map each visitor intent to a specific CTA.

**Prior related assets:**
- `docs/marketing/iterations/v2/audit-hero-for-marketing-clarity.md` — hero copy audit; same positioning gaps surface here (AI invisible above fold, four surfaces buried). Blog content can close these awareness gaps pre-click.
- `docs/marketing/iterations/v2/add-proof-strip-copy-plan.md` — proof strip copy; glossary pages can quote the same social proof stats and feed landing page conversions.
- `docs/marketing/iterations/v2/write-activation-checklist.md` — activation flow; tutorial-type posts (see items 11, 19) should mirror the activation steps.
- `docs/marketing/iterations/v2/add-email-capture-copy-variants.md` — email capture variants; each blog post should have an inline capture moment using the best-fit variant.

## Content Cluster Map

The 20 titles fall into five clusters. Each cluster owns a distinct audience intent and maps to the same core product but via a different entry angle.

```
Cluster A — Category Definition  (glossary, "what is X")
Cluster B — Pain-Point / DM Fatigue  (problem-aware, not solution-aware)
Cluster C — Use Case / Persona  (solution-aware, evaluating)
Cluster D — Comparison / Alternatives  (high-intent, comparison-ready)
Cluster E — Feature Deep-Dive  (existing user, referral, virality)
```

## Content Format Notes

- **Glossary items (A-cluster):** 600–900 words. Definition → context → example → live demo embed or screenshot → CTA. No fluff.
- **Pain-point pieces (B-cluster):** 800–1,200 words. Problem framing → consequences → solution comparison → Karte-specific walkthrough → CTA.
- **Comparison pieces (D-cluster):** Comparison table required. Feature parity matrix: links, AI chat, FAQ, import time, price, mobile. Karte wins on AI/interactivity; acknowledge where Linktree/Beacons win (ecosystem, monetization).
- **Tutorial pieces (C/E-cluster):** Step-by-step with screenshots. Numbered steps. Embed the `/sarthak` live demo. Exit intent capture for email (use "The 60-second setup guide" variant from `add-email-capture-copy-variants.md`).
